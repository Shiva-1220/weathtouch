⛅ Weather Forecasting Web App

A simple and responsive Weather Forecasting Web Application built with HTML, CSS, and JavaScript.
It uses the Open-Meteo API for fetching real-time weather data and provides a 5-day forecast with sunrise and sunset timings.

🚀 Features

✅ Search weather by city name
✅ Displays 5-day forecast (temperature, sunrise & sunset)
✅ Beautiful & responsive UI design
✅ Weather icons & conditions (sunny, cloudy, rain, storm, etc.)
✅ Error handling for invalid city names
✅ Built using vanilla JavaScript (no frameworks)

🛠️ Tech Stack

Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

API: Open-Meteo Weather API

Icons: Unicode + Custom styling

⚡ How It Works

Enter a city name in the search box.

The app fetches latitude & longitude using Open-Meteo Geocoding API.

The app fetches a 5-day forecast (max temp, min temp, sunrise, sunset, weather codes).

Results are displayed in card format with weather icons.

🔧 Installation & Setup

Clone this repository:

git clone git@github.com:Shiva-1220/weathtouch.git


Open the project folder and run it with Live Server in VS Code or simply open index.html in a browser.
